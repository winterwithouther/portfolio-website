PokeTeams
Github Demo Live

PokeTeams is an application where users are able to build out their own pokemon teams with just a few clicks. Users can search for specific pokemon through the naming convention, or if they don't have a certain pokemon in mind, they are always free to scroll the gallery and pick the one of user's choosing. With detailed information about each pokemon, user can add or remove from their team. Once the team is set, users can send their team to the public, where users can rate and get rated on their teams. Come build a team now with your favorite pocket monsters!

Technology: HTML, CSS, NativeJS

Wordle
Github Live

Wordle is a quick and easy brain exercise game where user's goal is solve the secret word. The game automatically chooses a secret word at random from the start and the user will be given 6 tries to guess the secret word. The game will give the user hints on which characters are in the right spot or included in the word. Inspired by the New York Times. Have fun figuring out the Wordle!

Technology: HTML, CSS, React, Bootstrap, Semantic UI

Yappin
Github Demo

Yappin is a full-stack application where users are to interact with each other through likes, posts, and comments. Upon signing up, users are greeted with a welcome message and immediately able to scroll posts. Users are also able to like and comment those posts. When nagivating to the header, users can post their own pictures and delete their posts. To the profile, we're given details about the user and the freedom to customize their profile picture, bio, and settings. Users are able to log out whenever they feel the need to stop yappin. Hop on and start Yappin!

Technology: NativeJS, React, HTML, CSS, Flask, SQLAlchemy, Bootstrap, Bcrypt
